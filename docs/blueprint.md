# **App Name**: TradeTrack

## Core Features:

- Stock Record Input: Allows users to manually add a new stock record with key details.
- Automatic Date and Time: Automatically fetches and populates the current date and time when creating a new stock record.
- Dynamic Stock Symbol Dropdown: Provides a dynamic dropdown menu to select a stock symbol when adding a record.
- Price Entry Fields: Input fields for entry price, stop loss, and target price, allowing manual price entry in Rupees.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), chosen to reflect focus, precision, and financial acumen.
- Background color: Light gray (#E0E0E0), a desaturated variant of the primary color to ensure a comfortable, neutral backdrop.
- Accent color: Bright fuchsia (#FF00FF), to highlight key actions and important data points.
- Body and headline font: 'Inter', a sans-serif font for a clean, modern interface suitable for displaying data.
- Use minimalist icons to represent different stock actions and data points.
- Clean, data-table driven layout, to clearly display all important records
- Subtle transitions when displaying records, sorting, and adding new stock information.
